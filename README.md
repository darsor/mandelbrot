Interactive Mandelbrot Viewer  
  
  
To compile and run:  
  
'''cmake .  
make  
./MandelViewer'''  
  
  
Controls Overview:  
H - help menu  
Q - quit  
S - save image  
R - reset view  
Scroll - zoom in/out  
Up/Down arrows - increase/decrease iterations  
Left/Right arrows - change the colors  
Numbers 1-7 (keyboard, not number pad) - change color scheme  
Click and Drag - ...click and drag
